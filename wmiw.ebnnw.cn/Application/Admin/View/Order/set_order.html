订单配置
正常流程
天  订单生成后，买家的支付期限。如果过期将扣除买家保证金，扣除佣金后自动转入卖家账户。
天  买家确认支付后，卖家发货时间期限。如果过期未发货，将扣除卖家保证金转入卖家账户。
天  卖家发货后，买家确认收货时间。如过期将自动确认收货。
天  买家确认收货或自动确认收货后，买家的评价期限。如过期将自动确认默认好评。
天  买家评价后，卖家评价买家的时间。如过期将自动好评。
如未按时支付或未按时发货网站抽取保证金的 %，剩余部分转入对方账户
退货流程
天  买家申请退货后，卖家同意退货，如过期将默认卖家为不同意退货。
天  卖家拒绝退货，买家申请调解，如过期将默认买家收货进入正常流程。
天  卖家同意退货后，买家发货，如过期将默认设置为买家已收货进入正常收货流程。
天  买家发货后，买家确认收货。如过期将自动确认收货，退货款项转入买家账户。
